Launch DeepLinks for oSpeech

Instructions:

  1. Download and install oSpeech on TestFlight:
    • Click on this link to install oSpeech on TestFlight (short invite).
  2. Open the app:
    • If you open the app without a deep link, it will use a full session (standard deep link).
    • If you open the app with a deep link from this page, it will have the correct exercises from that link.

XML Schema:

The XML file should follow the schema and guidelines outlined below:

    <?xml version="1.0" encoding="UTF-8"?>
    <trials>
        <trial>
            <name>Trial 01</name>
            <pluggedNose>false</pluggedNose>
            <challengeAmount>1</challengeAmount>
            <randomizeTrial>false</randomizeTrial>
            <phrases>
                <text>DESIRED_STRING:</text>
                <imagePath>IMAGE_PATH || NULL</imagePath>
                <videoPath>VIDEO_PATH || NULL</videoPath>
                <imageCompanionPath>IMAGE_PATH || NULL</imageCompanionPath>
            </phrases>
            <phrases>
                <text>DESIRED_STRING:</text>
                <imagePath>IMAGE_PATH || NULL</imagePath>
                <videoPath>VIDEO_PATH || NULL</videoPath>
                <imageCompanionPath>IMAGE_PATH || NULL</imageCompanionPath>
            </phrases>
            
        </trial>
    </trials>

Deep Link XML Integration:

The deep link can be constructed by appending the XML link to the following base URL:

ospeech://start?xml={XML_LINK}

Replace {XML_LINK} with the URL-encoded version of the desired XML link.

Deep Link Options:

  • Deep link to open on iOS (short session)
  • Deep link to open on iOS (full session session)
  • Deep link to open on iOS (multilingual session test)
